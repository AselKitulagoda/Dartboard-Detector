True Positive Rate
	Fraction of successfully detected faces out of all valid faces in an image

1) Difficult because you have to manually count the number of correct and missed hits.
2) Can always get 100% because you can detect everything in the image and so will always get all possible hits.
3) Implemented F1-Score 
	Precision = actual hits/number of detections
	Recall = actual hits/number of truths
	Recall is the same as TPR
